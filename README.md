# 김준우 게임 (Top-Down Action RPG)

Python과 Pygame으로 개발 중인 2D 탑다운 액션 RPG 프로젝트입니다.

## 실행 방법

1.  프로젝트 루트 디렉토리로 이동합니다.
2.  `run.bat` 파일을 실행합니다.
    ```bash
    run.bat
    ```

## 컨트롤

-   **이동:** WASD 또는 방향키
-   **점프:** Spacebar
-   **스킬:** Q
