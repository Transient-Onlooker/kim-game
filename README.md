# RPG (Top-Down Action RPG)

Python과 Pygame으로 개발 중인 2D 탑다운 액션 RPG 프로젝트입니다.

## 실행 방법

1.  프로젝트 루트 디렉토리로 이동합니다.
2.  `run.bat` 파일을 실행합니다.
    ```bash
    run.bat
    ```

## 컨트롤

-   **이동:** WASD 또는 방향키
-   **대쉬:** Spacebar
    -   이동 중(WASD)에는 해당 방향으로, 멈춰있을 때는 마우스 커서 방향으로 순간이동합니다. (무적 상태 포함)
-   **스킬:** Q
-   **조준 모드 전환:** TAB (자동 조준 / 수동 조준)
-   **수동 발사 모드 전환:** F (수동 조준 모드일 때만 작동, 클릭 발사 / 커서 자동 발사)
